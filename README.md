 About
This is the official website for Sabby Grand Hotel, a luxury hotel located in Murang'a County, Kenya.
The site was designed to give guests a premium online experience — showcasing the hotel's facilities, accepting booking enquiries, and providing directions.

 Features

 Parallax Scrolling — smooth, layered depth effects on all hero sections
 Fully Responsive — works on mobile, tablet and desktop
 Booking Form — connected to Formspree for real email delivery
 Google Maps — embedded directions to the hotel
 Bold & Modern Design — dark luxury theme with gold accents
 Real Hotel Photos — all images from the actual Sabby Grand Hotel

 How to Run Locally

Download or clone this repository:

bashgit clone https://github.com/YOUR_USERNAME/sabby-grand-hotel.git

Open the folder:

bashcd sabby-grand-hotel

Open SabbyGrandHotel_FINAL.html in your browser — that's it! No server needed.

 Contact Form Setup
The booking form uses Formspree for email delivery.
To set up your own:

Go to formspree.io and sign up
Create a new form
Replace the form action URL in the HTML:

html<form action="https://formspree.io/f/YOUR_CODE" method="POST">

 Deployment
This site is hosted for free on Netlify.
To deploy your own copy:

Go to netlify.com
Sign up free
Drag and drop the SabbyGrandHotel_FINAL.html file
Your site is live instantly!
